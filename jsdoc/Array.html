pairWith(field, arr) → {Array}

Pairs up an array of items with a plain array or variable by adding its values to a new field. If the argument is an array, it will be cycled through the items.

Parameters:
Name Type Description
field String The name of the new field
arr Array | * The array or object to put in the new field
Returns:
Array The paired array.
Examples

// typical use to pair up views with items
var experimentBlock = items.pairWith("view", "stimulus.ejs");
var arr = [];
for(var i = 0; i < 10; i++) 
  arr[i] = {"number": i};
arr.pairWith("parity", ["odd","even"]);
// result: [{number: 1, parity: "odd"}, {number: 2, parity: "even"}, 
// {number: 3, parity: "odd"}, {number: 4, parity: "even"}, 
// ... {number: 10, parity: "even"}]
